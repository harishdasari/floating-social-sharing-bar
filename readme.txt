== Description ==

Floating Social Sharing Bar is Social Sharing Counter Horizontal Bar which fixed and floats along with scrolling the page. 

This is Firstly Built for Blogger.com and converted to WordPress plugin. 
You can see it's Demo at [Demo Page](http://floatingsocialbar.blogspot.com/ncr/2012/06/sample-post-for-demo-blog.html). This Demo is on Blogger. But the Both the Plugins are same. 

== Installation ==

1. Upload `floating-social-sharing-bar` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Enable the Plugin in **Floating Social Bar** Options Page under the 'Settings' tab
4. Customize the Settings as you need and Click **Save Changes** Button
